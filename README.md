# caddie
A simple CAD tool for use in a browser


## Notes

### Types

- Shapes
- Materials

### Actions

- Create new shape
  - From template
  - Duplicate existing
- Delete shape
- Modify shape
  - Resize
    - xmin, xmax, ymin, ymax, zmin, zmax
  - Rotate
  - Translate
- Complex
  - Carve
  - Clip
- Undo/Redo
